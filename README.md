# Kuli Dashboard Service
Web service for Kuli Dashboard

## Getting started
Start with `node src/app.js`. Make sure you have mongodb running on your machine, and configs it on file `/src/configs/Mongo.js`

## Updates
This service using google SMTP to send mail to user. Please configs your own gmail on file `/src/configs/Mail.js`. Make sure your gmail is configured to can be used by third party application.
